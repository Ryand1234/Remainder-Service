Remainder Service

Remainder Service is a service in which you can add you content and it will send email to the users automatically.

Currently emails got sent to users at 5:12 AM, which will become configurable in future updates.

How to add notification

Hit this API with your content and it will add the content for cron job.

Data Format:

{ "to": "recivers email address", "content": "email content", "subject": "email subject", "date": "date of sending email }
